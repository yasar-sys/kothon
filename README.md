## ✨ Features

- 🔐 Anonymous message inbox (Firebase Firestore)
- 🌐 Hosted HTML form with dynamic usernames
- 🖼️ Bengali font styling (Siyam Rupali)
- 🔗 Instagram & QR integration
- 💌 Emotionally expressive animations
